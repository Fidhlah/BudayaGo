# cara run

> important: jangan lupa update .env kalau baru ganti api key atau kalau belum ada 

```
cd RAG_python
pip install -r requirements.txt
python ingest_data.py
```

kalau pake versi python yg spesifik, misal versi 3.12:

```
python -3.12 -m pip install -r requirements.txt
python -3.12 ingest_data.py
```
